## 📄 Estructura general

La página está construida en **HTML5** y organizada como una presentación dinámica con navegación por diapositivas.

---

## 🧱 Encabezado (`head`)

Incluye la configuración principal del documento:

- Codificación: `UTF-8`
- Compatibilidad: `IE=edge,chrome=1`
- Diseño responsive para dispositivos móviles
- Compatibilidad con aplicaciones móviles y Apple devices
- Título de la página: **Misiones del profe Edgar**
- Hoja de estilos externa: `style.css`
